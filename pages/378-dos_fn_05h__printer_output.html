DOS Fn 05H: Printer Output

                                                         Compatibility: 1.0+
 Expects: AH    05H
          DL    character to send to the Standard Printer Device
          ──────────────────────────────────────────────────────────────────
 Returns: none
          ──────────────────────────────────────────────────────────────────
    Info: Sends the character in DL to the Standard Printer Device
          (normally LPT1).

   Notes: ■ The DOS MODE command can redirect this output to a serial port.
