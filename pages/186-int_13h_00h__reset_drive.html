INT 13H 00H: Reset Drive

                                                              [PC] [XT] [AT]
 Expects: AH    00H
          ──────────────────────────────────────────────────────────────────
 Returns: AH    BIOS disk error code if CF is set to CY
          ──────────────────────────────────────────────────────────────────
    Info: Forces a recalibration of the diskette and hard disk controller.

          It places head 0 to track 0 of all drives.

          After an unexpected disk error, use this, then retry the
          operation.
