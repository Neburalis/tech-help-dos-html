INT 2fH 0103H: Cancel All Files

                                                         Compatibility: 3.0+
 Expects: AX    0103H
          ──────────────────────────────────────────────────────────────────
 Returns: none
          ──────────────────────────────────────────────────────────────────
    Info: Stops the current print job and removes all files from the print
          queue (only if PRINT.EXE is resident in memory).

See Also: INT 2fH 01xxH: DOS Print Spooler Functions
          INT 2fH: Multiplex Interrupt
