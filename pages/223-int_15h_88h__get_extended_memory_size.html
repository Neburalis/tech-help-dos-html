INT 15H 88H: Get Extended Memory Size

                                                                        [AT]
 Expects: AH    88H
          ──────────────────────────────────────────────────────────────────
 Returns: AX    amount of contiguous memory above 1MB, in 1K blocks.
          ──────────────────────────────────────────────────────────────────
    Info: This returns the total amount of memory above 1MB.  It is the
          same number as found in CMOS Memory addresses 17H-18H.
