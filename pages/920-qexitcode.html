ExitCode

                   exit code
────────────────────────────────────────────────
An 8-bit value furnished by any program when it
terminates via DOS fn 4fH (exit) or 31H
(terminate and go resident).
────────────────────────────────────────────────
■ DOS fn 4dH obtains the most recent exit code.
■ The IF ERRORLEVEL command tests the exit code.
