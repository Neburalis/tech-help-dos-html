DOS Fn 6601H: Query Active Code Page

                                                         Compatibility: 3.3+
 Expects: AX    6601H
          ──────────────────────────────────────────────────────────────────
 Returns: AX    error code if CF is set to CY
          BX    current active code page (as set by the user)
          DX    system code page (a setup at boot time)
          ──────────────────────────────────────────────────────────────────
    Info: Obtains the code page ID for the current active code page and the
          default system code page, as set during system startup.

   Notes: ■ The system code page is set via the COUNTRY= command in
            CONFIG.SYS.

See Also: fn 6602H (activate code page)
