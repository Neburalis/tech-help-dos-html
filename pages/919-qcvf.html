CVF

                             CVF
                   Compressed Volume File
────────────────────────────────────────────────────────────
A CVF is a file with a name such as DBLSPACE.000 or
DBLSPACE.001; it contains all of the files and file-system
bookkeeping information for a DoubleSpace compressed disk.
────────────────────────────────────────────────────────────
Before a CVF can be accessed as a disk, it must be mounted;
that is, the DoubleSpace support must recognize it and assign
it a drive letter.
────────────────────────────────────────────────────────────
