INT 2fH 1602H: Get Enh386 Windows Entry-Point Address

                                                      Compatibility: Win 3.x
 Expects: AX    1602H
          ──────────────────────────────────────────────────────────────────
 Returns: ES:DI far address of Windows entry-point procedure
          ──────────────────────────────────────────────────────────────────
    Info: This is an obsolete call used with Windows/386 ver 2.x.  It is
          maintained for compatibility.  Check the DDK if you need this fn.
