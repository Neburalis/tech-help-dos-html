INT 1eH: Diskette Parameter Pointer

 This vector (0:0078) is a pointer to a table of diskette parameters that
 BIOS the diskette service INT 13H uses for a variety of diskette
 operations.

 This table includes values such as "head-settle time" and "step rate time"
 which have a tremendous effect on the speed of diskette I/O.

 It is possible to change some facets of diskette operation by creating a
 RAM-resident version of this table and modify selected parameters.  DOS
 creates a modified version of this table to improve diskette performance.
 Oddly, DOS restores the normal (typically slower) values when it restores
 control to the application program.

 See Diskette Parameter Table for a layout of the table.
