CVF Region: RootDir

 The RootDir is a 32-sector (16K=16,384 bytes) region formatted as a
 directory.  It is located directly after the end of the FAT; its
 location can be calculated as MdBpbRec.wMdResSects+wMdRootStart sectors
 from the start of the CVF.

 It contains 512 DirEntryRecs.  These are identical in every way to
 standard directory entries.  DoubleSpace does not appear to use any of the
 "reserved" bytes in the directory entry structure.
