Network and File-Sharing Functions

 INT 2fH 1000H see if file-sharing support is installed

 440bH set sharing retry count
 5c00H lock part or all of a file
 5c01H unlock part or all of a file

 INT 2fH 1000H see if network support is installed

 4409H see if drive is local or remote
 440aH see if file or device is local or remote

 5e00H get machine name (network node name)
 5e02H set network printer setup
 5e03H query network printer setup

 5f02H get assign-list entry (get index number needed for other fns)
 5f03H make network connection (associate drive ID with remote drive)
 5f04H delete network connection
