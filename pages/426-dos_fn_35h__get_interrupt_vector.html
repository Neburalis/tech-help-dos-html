DOS Fn 35H: Get Interrupt Vector

                                                         Compatibility: 2.0+
 Expects: AH    35H
          AL    interrupt number (00H to 0ffH)
          ──────────────────────────────────────────────────────────────────
 Returns: ES:BX address of the interrupt handler
          ──────────────────────────────────────────────────────────────────
    Info: Returns the value of the interrupt vector for INT (AL). It loads
          BX with 0000:[AL*4], and ES with 0000:[(AL*4)+2].

See Also: Fn 25H (set INT Vector)
