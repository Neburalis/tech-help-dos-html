INT 2fH 1704H: Query Size of Clipboard Data

                                                    Compatibility:  WIN 3.0+
 Expects: AX    1704H
          DX    WinOldApp-supported clipboard data format code
          ──────────────────────────────────────────────────────────────────
 Returns: DX:AX size, in bytes of the data, including any headers.
                DX is the high word and AX is the low word.
                0000:0000 means no data in format DX was found.
          ──────────────────────────────────────────────────────────────────
    Info: This obtains the size of the data that is currently in the
          clipboard.  Equivalent to GetClipboardDataSize.

          Only a few data types are supported.  See Clipboard Data Formats.

          On CF_BITMAP-formatted data (DX=0002H), the returned size will
          include the size of the 24-byte header.
