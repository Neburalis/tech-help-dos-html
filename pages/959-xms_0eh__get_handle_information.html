XMS 0eH: Get Handle Information

                                                     Compatibility: XMS 2.0+
 Expects: AH    0eH
          DX    XMS handle (as obtained via XMS 09H)
          ──────────────────────────────────────────────────────────────────
 Returns: AX    status: 0000H = successful
                        0001H = error
          BH    current lock count
          BL    current number of free XMS handles
          DX    size of the block, in K-bytes
          ──────────────────────────────────────────────────────────────────
    Info: This fn obtains information about an extended memory block.

          To get the 32-bit address of the block, use XMS 0cH (lock XMB).
