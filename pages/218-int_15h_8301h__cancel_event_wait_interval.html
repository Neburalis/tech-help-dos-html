INT 15H 8301H: Cancel Event Wait Interval

                                                                        [AT]
 Expects: AX    8301H
          ──────────────────────────────────────────────────────────────────
 Returns: AX    8300H
          CF    NC (0) no error
                CY (1) error; fn is busy
          ──────────────────────────────────────────────────────────────────
    Info: Cancels the most recent timer interval set via INT 15H 8300H.
