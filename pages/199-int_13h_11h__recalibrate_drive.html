INT 13H 11H: Recalibrate Drive

                                                                   [XT] [AT]
 Expects: AH    11H
          DL    drive: 80H-81H=hard disk 0-1
          ──────────────────────────────────────────────────────────────────
 Returns: AH    BIOS disk error code if CF is set to CY
          ──────────────────────────────────────────────────────────────────
    Info: This positions head 0 head of the specified hard drive over
          track 0.

   Notes: ■ This is automatically performed by reset fns 00h and 0aH.
