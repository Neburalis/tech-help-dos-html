◄Print Topic►

┌──╖
│F5║ prints the text of the current topic.
╘══╝ By default,text is sent 'raw' to the PRN device.

Printer options can be set by using a text editor to
modify XVIEW.INI (the Viewer configuration file).
Options include:

 prnName=PRN            (device or FILENAME.EXT)
 prnLines=59            (lines per page)
 prnSetup=<any_text>    (printed before each topic)
 prnFilterBoxChars=1    (e.g., print +--+, not ┌──┐)
