hyperlink

A hyperlink is a word or phrase which is
connected to a destination page.

┌──┤page├──┐╔═►┌──┤page├──┐╔═►┌──┤page├──┐
│text text │║  │text text │║  │text text │
│   link═══╪╝  │ text text│║  │ text text│
│ text text│   │   link═══╪╝  │text text │
│text text │   │ text text│   │ text text│
└──────────┘   └──────────┘   └──────────┘
It may also be connected to a viewer
function such as "Exit" or "BackTrack" or
to a popup definition, such as this one.
