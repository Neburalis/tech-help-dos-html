INT 10H 0fH: Query Current Video Info

                                                          Compatibility: All
 Expects: AH    0fH
          ──────────────────────────────────────────────────────────────────
 Returns: AL    current video mode
          AH    width of screen, in character columns
          BH    current active video page (0-based)
          ──────────────────────────────────────────────────────────────────
    Info: This obtains a certain data regarding the current video mode.

          Use INT 10H 12H (special fns) and
          see EGA/VGA Data Areas to get info on EGA modes.

          Use  INT 10H 1bH (get functionality/state info) for VGA modes.

          Use INT 10H 4f03H (get SVGA info) to get info on VESA modes

See Also: INT 10H: Video Services
