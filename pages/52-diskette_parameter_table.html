Diskette Parameter Table

 A series of these 11-byte structures can be found at the vector address of
 INT 1eH (the 4-byte address found at 0:0078).  The list is also known as
 the "Disk Base Table."

 It specifies a variety of critical variables for diskette drives only.
 The table is set to default values by the ROM-BIOS and may be modified by
 DOS to improve diskette performance or increase battery life on a laptop.
