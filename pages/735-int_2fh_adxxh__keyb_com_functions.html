INT 2fH adxxH: KEYB.COM Functions

                                                         Compatibility: 3.0+
     See: ad80H (query KEYB.COM version)

          ad81H (set KEYB.COM active code page)

          ad82H (set KEYB.COM country flag)

          ad83H (query KEYB.COM country flag)

 KEYB.COM is a DOS program which provides national language support for
 typing special characters on various keyboards.

 These functions are available only when the KEYB.COM command has been
 executed and is resident in memory.  Use INT 2fH ad80H to see if it is
 loaded.
