INT 33H 000eH: Disable Lightpen Emulation

 Expects: AX    000eH
          ──────────────────────────────────────────────────────────────────
 Returns: none
          ──────────────────────────────────────────────────────────────────
    Info: This tells the mouse support to stop emulating a light pen.  You
          would use this to keep the mouse movement from interfering with
          the operation of an attached light pen.

          When enabled via INT 33H 000dH, the mouse support apparently
          intercepts INT 10H 04H.

See Also: INT 33H: Mouse Support
          Interrupts and BIOS Services
