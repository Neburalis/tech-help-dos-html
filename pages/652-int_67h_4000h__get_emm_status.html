INT 67H 4000H: Get EMM Status

                                                     Compatibility: EMS 3.2+
 Expects: AX    4000H
          ──────────────────────────────────────────────────────────────────
 Returns: AH    EMM status
          ──────────────────────────────────────────────────────────────────
    Info: Obtain the current status of the Expanded Memory Manager.

          Use this only after establishing the presence of EMM support (see
          Expanded Memory Specification for details).
