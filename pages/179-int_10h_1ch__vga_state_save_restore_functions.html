INT 10H 1cH: VGA State Save/Restore Functions

                                                          Compatibility: VGA
     See: 1c00H     VGA (query size of save buffer)

          1c01H     VGA (save current video state)

          1c02H     VGA (restore video state from saved data)

 These functions are put in place by ROM-Scan when a VGA▲ card is installed
 and are built into the standard ROMS for PS/2 systems.  They are not
 available on EGA▲, CGA▲ or MDA▲ systems.
