TSR

        memory resident (TSR) programs
───────────────────────────────────────────────
TSR stands for Terminate and Stay Resident; it
refers to a program which leaves all or part of
itself in memory after it exits to DOS.
───────────────────────────────────────────────
■ DOS fn 31H makes a program RAM-resident
■ INT 27H is the old way to go TSR.
