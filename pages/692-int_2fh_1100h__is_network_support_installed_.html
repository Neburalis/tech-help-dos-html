INT 2fH 1100H: Is Network Support Installed?

                                                         Compatibility: 3.0+
 Expects: AX    1100H
          ──────────────────────────────────────────────────────────────────
 Returns: AL    status: 00H = is not installed
                        ffH = is installed
          ──────────────────────────────────────────────────────────────────
    Info: Checks to see if network support code has been installed and is
          resident in memory.

See Also: INT 2fH: Multiplex Interrupt
          DOS Functions
