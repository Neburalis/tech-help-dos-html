INT 2fH 0600H: Is Assign.Com Installed?

                                                         Compatibility: 3.0+
 Expects: AX    0600H
          ──────────────────────────────────────────────────────────────────
 Returns: AL    status: 00H = is not installed
                        ffH = is installed
          ──────────────────────────────────────────────────────────────────
    Info: Checks to see if the DOS Assign.Com program has been executed and
          is resident in memory.

See Also: INT 2fH: Multiplex Interrupt
          DOS Functions
