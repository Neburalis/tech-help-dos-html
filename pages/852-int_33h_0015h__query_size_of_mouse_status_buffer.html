INT 33H 0015H: Query Size of Mouse Status Buffer

 Expects: AX    0015H
          ──────────────────────────────────────────────────────────────────
 Returns: BX    size of buffer needed in INT 33H 0016H (save mouse status)
          ──────────────────────────────────────────────────────────────────
    Info: Use this function before calling INT 33H 0016H.  It returns the
          size of the status-saving buffer you will need in that call.

See Also: INT 33H: Mouse Support
