INT 10H 1114H: Load and activate ROM 8x16 Character Set

                                                          Compatibility: VGA
 Expects: AX    1114H
          BL    font block to load (VGA: 0-7)
          ──────────────────────────────────────────────────────────────────
 Returns: (none)
          ──────────────────────────────────────────────────────────────────
    Info: Redefines the characters on VGA cards to use the standard 16-
          scanline-high text-mode font, exactly as described for
          INT 10H 1104H, except that it takes the additional steps needed
          to activate the font.

See Also: INT 10H 11H: EGA/VGA Character Generator Functions
