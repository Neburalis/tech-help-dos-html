INT 10H 00H: Set Video Mode

                                                          Compatibility: All
 Expects: AH    00H
          AL    video mode number (optionally, +80H for don't clear data)
          ──────────────────────────────────────────────────────────────────
 Returns: (none)
          ──────────────────────────────────────────────────────────────────
    Info: This initializes the video hardware to display in the specified
          video mode.

          See Video Modes for a list of standard video modes.

          See Video Memory Layouts for info on accessing screen memory.

See Also: INT 10H: Video Services
          VESA/SVGA BIOS Services
