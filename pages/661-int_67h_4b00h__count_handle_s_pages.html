INT 67H 4b00H: Count Handle's Pages

                                                     Compatibility: EMS 3.2+
 Expects: AX    4b00H
          DX    EMM Handle (as obtained via INT 67H 4300H)
          ──────────────────────────────────────────────────────────────────
 Returns: AH    EMM status (0=successful)
          ──────────────────────────────────────────────────────────────────
    Info: Returns number of EMS pages owned by a handle.
