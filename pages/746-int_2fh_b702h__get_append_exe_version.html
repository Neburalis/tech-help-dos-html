INT 2fH b702H: Get APPEND.EXE Version

                                                         Compatibility: 4.0+
 Expects: AX    b702H
          ──────────────────────────────────────────────────────────────────
 Returns: CF    Carry Flag clear (NC) if no error
          AX    ffffH means this version compatible with DOS 4.0+
          ──────────────────────────────────────────────────────────────────
    Info: Returned values other than ffffH indicate some version older than
          that of DOS 4.0.

          The interface to APPEND.EXE has not changed, up to and including
          DOS 6.0.
