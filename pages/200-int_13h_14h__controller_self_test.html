INT 13H 14H: Controller Self-Test

                                                                   [XT] [AT]
 Expects: AH    14H
          ──────────────────────────────────────────────────────────────────
 Returns: AH    BIOS disk error code if CF is set to CY
          ──────────────────────────────────────────────────────────────────
    Info: This forces the hard disk controller to perform an internal self-
          test.  Result is returned in AH.
