DOS Fn 5d0aH: Set Extended Error

                                                         Compatibility: 5.0+
 Expects: AX    5d0aH
          DS:SI address of an ErrorInfoRec containing error info for fn 59H
          ──────────────────────────────────────────────────────────────────
 Returns: none
          ──────────────────────────────────────────────────────────────────
    Info: Sets the error class, location, suggested action, and other
          information that will be returned by the next call to fn 59H (get
          extended error info).
