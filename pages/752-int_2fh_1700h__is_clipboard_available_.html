INT 2fH 1700H: Is Clipboard Available?

                                                     Compatibility: WIN 3.0+
 Expects: AX    1700H
          ──────────────────────────────────────────────────────────────────
 Returns: AX    1700H = Clipboard fns are not available
                else  = AL= major version number
                        AH= minor version number
          ──────────────────────────────────────────────────────────────────
    Info: Use this to see if you are running in a DOS session under Windows
          and to see if clipboard support is available.  Equivalent to
          IdentifyWinOldApVersion.

See Also: INT 2fH 17xxH: Windows Clipboard Functions
          INT 2fH: Multiplex Interrupt
