INT 10H 1112H: Load and Activate ROM 8x8 Character Set

                                                      Compatibility: EGA VGA
 Expects: AX    1112H
          BL    font block to load (EGA: 0-3; VGA: 0-7)
          ──────────────────────────────────────────────────────────────────
 Returns: (none)
          ──────────────────────────────────────────────────────────────────
    Info: Redefines the characters on EGA/VGA cards to use the 8-scanline-
          high text-mode font, exactly as described for INT 10H 1102H,
          except that it takes the additional steps needed to activate the
          font.

See Also: INT 10H 11H: EGA/VGA Character Generator Functions
