INT 0dH: IRQ 5 LPT2 / General Protection Fault

  INT 0fH is a hardware-generated interrupt (IRQ 7) forced by LPT1 when it
  is ready to receive a character (but only when the printer controller has
  been programmed to generate the interrupt).

█▌CPU Exception Interrupt▐█
  286+ computer execute INT 0dH when the CPU is in protected mode and it
  detects any memory access outside of the areas available to a task.
