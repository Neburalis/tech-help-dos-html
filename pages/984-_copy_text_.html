◄Copy Text►

┌─────╖ ┌──╖
│Shift║+│F5║ copies lines of text from the current
╘═════╝ ╘══╝ file into a file named PASTE.TXT.

1) Press [Shift+F5] and use the arrow keys to move to the
   first line that you want to copy.
2) Press [Enter] and use the arrows to mark the desired block.
3) Press [Enter] again to write the marked text to the file.

─╔═╝────────────────────────────────────── Using the Mouse ───
┌╩─┐ Press [Shift+F5].
│▌▐│ Move the mouse pointer to the desired first line.
│  │ Press the [Left Button] and drag to mark the block.
└──┘ Release the button and click [OK] to save.
